### Типы полей в MySQL

##### Наиболее часто применяемые:

* VARCHAR 
  : для хранения логинов, паролей, заголовков статей, ФИО и т.д.

* TEXT
  : для хранения текстовых данных

* INT
  : используется наиболее часто для хранения ID

* DATETIME
  : для хранения даты и времени
